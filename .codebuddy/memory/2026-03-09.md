## ThemeStudio 布局偏移修复

修复了点击进入 ThemeStudio 后存在布局偏移的问题，根因是全局样式污染：

**修复内容：**
1. `index.css` — body 的 `display:flex` + `place-items:center` 改为仅 `.home-page` 时生效（用 `:has()` 选择器）
2. `index.css` — 全局 `button` 样式限定到 `.home-page button`，不再污染 Studio 内部按钮
3. `App.css` — `#root` 的 `max-width:1280px` / `margin:0 auto` / `padding:2rem` 改为 `#root:has(.home-page)` 条件生效
4. `ThemeStudio.module.css` — 改用 `position:fixed` 全屏覆盖 + `all:unset` 重置内部 button + `z-index:100`
5. 各子组件 CSS 更新为完全自包含样式（不依赖全局 button 样式）
6. `App.tsx` — 首页容器添加 `className="home-page"` 作为样式作用域标记
