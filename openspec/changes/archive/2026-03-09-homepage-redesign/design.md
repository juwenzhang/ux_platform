# Design: 首页重构方案

## 1. 整体布局

采用经典的 SaaS 门户首页布局，从上到下依次为：

```
┌─────────────────────────────────────────┐
│  Header (导航栏)                         │
│  [Logo] [产品名]        [主题切换] [GitHub] │
├─────────────────────────────────────────┤
│  Hero 区域                                │
│  [大标题 + Slogan + CTA 按钮]             │
│  [功能预览卡片 / 动画演示]                 │
├─────────────────────────────────────────┤
│  功能介绍区域                              │
│  [特性卡片 1] [特性卡片 2] [特性卡片 3]    │
├─────────────────────────────────────────┤
│  产品矩阵 (未来扩展预留)                   │
│  [Theme Studio] [Tool B] [Tool C]        │
├─────────────────────────────────────────┤
│  Footer                                   │
└─────────────────────────────────────────┘
```

## 2. 详细设计

### 2.1 Header (导航栏)

- **左侧**: 项目 Logo + 名称 "OpenSpec"（或当前项目名）
- **右侧**: 
  - ThemeSwitcher 主题切换组件
  - GitHub 链接按钮（可选）

### 2.2 Hero 区域

- **主标题**: "低代码主题配置平台" 或类似
- **副标题/Slogan**: 一句话介绍核心价值
- **CTA 按钮**: "Theme Studio" 跳转 Theme Studio
- **功能预览**: 
  - 放置一个迷你的主题预览组件
  - 展示几个主题色卡片、示例按钮等
  - 让用户直观感受主题配置效果

### 2.3 功能介绍区域

展示 3-4 个核心特性卡片：
1. **可视化配置** - 无需编码，通过 UI 配置主题
2. **实时预览** - 修改即时生效，所见即所得
3. **多主题支持** - 亮色/暗色主题，一键切换
4. **JSON 导出** - 标准格式导出，便于集成

每个卡片包含：图标 + 标题 + 简短描述

### 2.4 产品矩阵 (预留)

- 展示已有的低代码工具入口卡片
- 每个卡片: 图标 + 名称 + 简述 + "进入" 按钮
- 预留 2-3 个空位，未来填充其他工具

### 2.5 Footer

- 版权信息
- 技术栈标识 (Vite + React)

## 3. 目录结构

将首页组件独立到 `components/Home/` 目录：

```
src/components/
├── Home/
│   ├── Home.tsx           # 首页主组件
│   ├── Home.module.css    # 样式文件
│   ├── Hero.tsx           # Hero 区域
│   ├── Hero.module.css
│   ├── FeatureCard.tsx     # 功能卡片组件
│   ├── FeatureCard.module.css
│   ├── ProductCard.tsx    # 产品卡片组件
│   └── ProductCard.module.css
```

## 4. 技术实现

### 4.1 组件拆分策略

- `Home.tsx` - 页面容器，负责整体布局
- `Hero.tsx` - 顶部区域，可复用到其他页面
- `FeatureCard.tsx` - 功能卡片，props 驱动内容
- `ProductCard.tsx` - 产品卡片，用于产品矩阵

### 4.2 样式策略

- 使用 CSS Modules (`.module.css`)
- 保持现有 Design Token 风格一致
- 支持暗色主题（通过 CSS Variables）

### 4.3 路由调整

- `#/home` - 首页（默认）
- `#/studio` - Theme Studio
- 可扩展: `#/products/tool-name`

## 5. 验收标准

- [ ] 首页正确显示品牌信息（Logo + 名称）
- [ ] Hero 区域包含标题、Slogan、CTA 按钮
- [ ] 功能预览区展示主题效果
- [ ] 3-4 个功能介绍卡片
- [ ] 产品矩阵区域显示入口卡片
- [ ] 点击 CTA 按钮可跳转到 Theme Studio
- [ ] 支持主题切换（亮/暗色）
- [ ] 目录结构符合要求（css + tsx 同目录）
- [ ] 构建无错误
