# Proposal: 首页重构 - 低代码平台入口

## 背景

当前首页是 Vite + React 模板的默认页面，仅包含：
- Vite 和 React 的 logo
- 简单的计数器示例
- 跳转到 Theme Studio 的按钮

这个首页存在以下问题：
1. **缺乏品牌识别** - 没有项目名称、Logo 或视觉标识
2. **功能介绍缺失** - 用户不知道这个平台能做什么
3. **扩展性差** - 未来会增加其他低代码工具，但首页没有预留入口
4. **视觉简陋** - 没有美感，无法展示平台的专业性

## 目标

将首页重构为 **低代码平台门户首页**，实现：

1. **清晰的品牌标识** - 项目名称、Logo、简短的 Slogan
2. **功能介绍区域** - 展示核心功能（主题配置特点和）的能力
3. **实时功能预览** - 展示 Theme Studio 的实际效果，让用户快速了解功能
4. **多产品入口** - 预留其他低代码工具的入口卡片（暂不实现，但预留位置）
5. **目录结构优化** - 将 Home 组件的 CSS 和 TSX 放在同一目录下

## 收益

- 用户进入首页即可了解平台能力，降低学习成本
- 专业的首页设计提升平台可信度
- 为后续扩展其他低代码工具奠定基础
- 代码结构更清晰，维护性更好
